### 1.0.0
*Release Date 13th June 2026*

* Initial public release of WP Cache Autopilot after extended production use on real WordPress sites, including large installations with more than 1,200 pages.
* Added a unified cache freshness lifecycle from change detection through targeted invalidation, adapter-based purge, and paced warmup.
* Added targeted invalidation for content, archives, taxonomies, comments, metadata, and presentation-related changes.
* Added queue-based warmup for affected pages after purge.
* Added integrations for supported WordPress cache plugins.
* Added settings, logging, diagnostics, and developer extension points.
* Added Free/PRO-ready internal structure while keeping the free plugin fully functional on its own.
* Preserved historical changelogs for the former Cache Invalidator and Cache Warmup engines.
